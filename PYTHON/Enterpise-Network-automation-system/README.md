# Enterprise Network Automation System (ENAS)

A minimal, interactive Python CLI tool to automate VLAN operations (CRUD) on MikroTik RouterOS via SSH using Netmiko.

## 📊 Architecture & Demo Slides
Detailed project presentation and before-after screenshots are available in the **[Python automated.pdf]*** file inside this repository.

```bash
